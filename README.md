# Baseball Analytics App

This project is an interactive baseball statistics application built with Python and Streamlit.

## Purpose

The app analyzes historical MLB player statistics to help identify performance trends, compare players, and evaluate potential breakout or declining players.

## Features

- Historical batting statistics explorer
- Career totals aggregator
- Player comparison tool
- Trend analysis over multiple seasons
- Player valuation and performance indicators

## Tools Used

- Python
- pandas
- NumPy
- Streamlit
- Matplotlib

## Current Status

This project is currently in development. I am continuing to add predictive modeling and machine learning features.
